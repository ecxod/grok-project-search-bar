# Grok Project Search Bar
This Javascript add a search bar to the project list on grock.com

Get Yourself a Code Injector Add-on ( we recomand [Code Injector by Saba](https://addons.mozilla.org/en-US/firefox/addon/codeinjector/)

Load the following javascript into the Code Injector and visit https://grok.com/project
